### 转换命令

```bash
git clone --depth 1 https://github.com/lilinji/English.git tmp/lilinji/English
npm run convert:pep2019
```

输出：`public/data/textbooks/pep2019-*.json` + `index.json`。

### 单元划分说明

源 xlsx **无 Unit 列**。`scripts/convert-pep-2019.mjs` 按课本词表**起始词**切分：

| 册 | 单元结构 |
|----|----------|
| **必修第一册** | **Welcome Unit · 预备单元** + Unit 1～5（有标题） |
| 必修第二、三册 | Unit 1～5（有标题） |
| 选择性必修第一册 | Unit 1～5（有标题） |
| 选择性必修二～四 | 暂均分 5 单元（标题为 Unit 1～5） |

必修一预备单元约 57 词（exchange → revise），对应课本 Welcome Unit。
