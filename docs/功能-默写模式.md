# 功能实现：默写模式

> 日期：2026-07-17  
> 需求：单词列表增加「默写」按钮；开启后隐藏英文单词与音标，**只显示中文**。  
> 审查修复：tooltip 剧透、跟读面板/底栏、空列表退出、缓存单测等（见下）。

---

## 行为

| 区域 | 默写关闭 | 默写开启 |
|------|----------|----------|
| 单词列表 | 英文 + 音标 + 中文 | **仅中文** |
| 「英」按钮 title | `英：apple` | `听发音（不显示拼写）`（不剧透） |
| 跟读大字面板 | 英文 + 中文 | **「？」+ 中文** |
| 跟读底栏 | 英文 | **中：释义**（完成时显示「完成」） |
| 开关按钮 | 始终可见（含空列表） | 可「退出默写」 |

- 仍可点「英」听音自检。  
- `dictationMode` 写入 `le_selection_v1`，仅 `=== true` 为开启。

---

| 文件 | 说明 |
|------|------|
| `src/App.tsx` | 列表/面板/底栏；工具栏始终显示开关 |
