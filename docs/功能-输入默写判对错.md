## 行为

| 项 | 说明 |
|----|------|
| 触发 | **默写开启**且列表有词时，每卡显示输入框 |
| 提交 | 点「检查」或 Enter；**空输入按钮 disabled**，不静默失败 |
| 判对 | `trim` + 小写 + 多空格压单空格后与 `word.en` 全等 |
| 严格点 | **不**统一弯引号/去标点（须与词库写法一致） |
| 对 | 绿色 +「✓ 正确」 |
| 错 | 红色 +「✗ 正确答案：…」；`aria-invalid` |
| 重试 | 清空该词输入与结果 |
| 统计 | `已检 x · 对 y · 错 z`（按 `displayWords` 下标项） |
| 状态 key | `listItemSpellKey(index, en, zh)`，同词多条不串写 |
| 重置 | 列表**内容/顺序**签名变化、退出默写 → 清空；catalog 仅换引用不清 |
| 持久化 | session 内存 |
| 后端 | 无 |
